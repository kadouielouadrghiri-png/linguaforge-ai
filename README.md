## ✨ المميزات

- **AI Tutor** شخصي يتكيف مع مستواك ومجال عملك
- **محادثات صوتية** واقعية مع تصحيح النطق الفوري
- **Role-Play** احترافي (مقابلات عمل، اجتماعات، مفاوضات، عروض تقديمية)
- **Writing Assistant** يكتب ويصحح الإيميلات، التقارير، والمنشورات
- **مسارات تعلم مخصصة** (Business, Technical, Medical, Legal...)
- **Gamification** + تحديات يومية + نظام Streaks
- **Offline Mode** قوي
